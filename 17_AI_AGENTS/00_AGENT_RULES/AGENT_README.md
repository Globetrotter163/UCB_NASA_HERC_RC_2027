# Agent Operating Model

## Required flow

1. Read `AGENTS.md`, this folder and the task-specific context pack.
2. Identify authoritative inputs and missing information.
3. Work only in allowed paths.
4. Put output under `17_AI_AGENTS/03_AGENT_OUTPUTS/`.
5. State sources, assumptions, uncertainty and unresolved conflicts.
6. Request human review.
7. A human integrates accepted content into the official document.

## Prohibitions

- No destructive operations.
- No automatic technical approvals.
- No invented measurements, test results or citations.
- No silent resolution of conflicting sources.
- No marking deliverables as submitted.
